# ABC E-commerce Sales Dashboard

## Project Overview
This project focuses on analyzing sales data from an e-commerce business (ABC E-commerce) to understand sales performance, customer behavior, and product trends.

## Objective
- Analyze overall sales performance
- Identify top-selling products and categories
- Track monthly revenue trends
- Support business decision-making using data

## Tools Used
- Power BI
- Excel
- Data Visualization techniques

## Key Insights
- Identified top-performing products contributing the highest revenue
- Observed seasonal sales trends across months
- Found customer segments with higher purchase frequency

## Dashboard Features
- Total Sales & Revenue KPIs
- Category-wise and product-wise sales analysis
- Monthly sales trends
- Interactive filters for deeper analysis

## Outcome
Created an interactive dashboard that helps stakeholders quickly understand business performance and make data-driven decisions.
